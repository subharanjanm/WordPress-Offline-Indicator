WordPress Offline Indicator uses \'offline.js\' library to automatically alert users when they\'ve lost internet connectivity, like Gmail. As soon as the browser senses the loss of the internet connection, a message is issued on the WordPress backend, giving your WP users the information that – they have lost the connection.

== Description ==
WordPress Offline Indicator uses \'offline.js\' library to automatically alert users when they\'ve lost internet connectivity, like Gmail. As soon as the browser senses the loss of the internet connection, a message is issued on the WordPress backend, giving your WP users the information that – they have lost the connection.

== Installation ==
1. Upload \"wordpress-offline-indicator\" zip file to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.

== Frequently Asked Questions ==
= Does it display alerts on front-end of the website ? =
No. This only displays messages on WordPress back-end.
